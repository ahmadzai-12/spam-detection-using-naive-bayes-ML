# spam-detection-using-naive-bayes-ML
A machine learning-based web application for detecting spam messages using the Naive Bayes algorithm, built with Python and Flask.
# 📧 Spam Detection Using Naive Bayes

This project is a **machine learning-based spam detection system** that classifies messages or emails as *Spam* or *Not Spam (Ham)* using the **Naive Bayes algorithm**. The system is developed using **Python** for model training and **Flask** for building a lightweight and interactive web interface.

## 🚀 Project Overview

Spam messages are a major issue in digital communication. This project provides an efficient solution by applying **Natural Language Processing (NLP)** and **Naive Bayes classification** to automatically detect and filter spam content.

## 🧠 Technologies Used

* **Python** – Core programming language
* **Flask** – Web framework for building the user interface
* **Scikit-learn** – Machine learning library for training the model
* **Pandas & NumPy** – Data processing and analysis
* **TF-IDF / Count Vectorizer** – Text feature extraction

## ⚙️ Features

* Classifies text messages as **Spam or Ham**
* Simple and user-friendly **web interface**
* Fast and efficient **Naive Bayes model**
* Real-time prediction using Flask API
* Preprocessing of text data (cleaning, tokenization)

## 🔍 How It Works

1. Text data is collected and preprocessed
2. Features are extracted using vectorization techniques
3. A **Naive Bayes classifier** is trained on labeled data
4. The model predicts whether a new message is spam or not
5. Flask provides a web interface for user interaction

## 📊 Use Cases

* Email spam filtering
* SMS spam detection
* Text classification tasks
* NLP learning projects

## 🖥️ How to Run

1. Clone the repository
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the Flask app:

   ```bash
   python app.py
   ```
4. Open in browser:

   ```
   http://127.0.0.1:5000
   ```
